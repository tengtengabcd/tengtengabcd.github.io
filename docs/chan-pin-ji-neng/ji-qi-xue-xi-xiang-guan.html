    机器学习相关

    藤藤同学 3/26/2022 机器学习

    机器学习：一个计算机程序在一些任务上的性能，会随着计算机模型训练后，随着训练经验而提高，这个过程就是机器学习。

    基本的机器学习算法：

    • 线性回归算法 —预测
    • k-平均算法 K-Means—聚类算法
    • 降维算法—得到一组“不相关”主变量
    • 支持向量机算法—二分类
    • 决策树算法 —二分类
    • 随机森林算法 —二分类
    • 朴素贝叶斯算法—二分类
    • k-近邻算法—多分类
    • 逻辑回归算法—二分类、多分类

    1、线性回归算法
    简单线性回归，只有 1 个自变量；多变量回归（multiple regression)，至少两组以上自变量。
    对已知的多个相关数据进行处理，建立模型，形成一个函数关系，利用已知数据对结果进行预测。例如：对房价的预测。

    # 2、支持向量机算法（二分类）

    监督学习算法。SVM模型将实例表示为空间中的点，将使用一条直线分隔数据点。仅直接适用于两类任务，应用将多类任务需要减少到几个二元问题。

    # 3、k-近邻算法-KNN （多分类问题）

    用最近的邻居（k）来预测未知数据点。计算待标记样本和数据集中每个样本的距离，取距离最近的k个样本。待标记的样本所属类别就由这k个距离最近的样本投票产生。比较依赖训练数据集

    # 4、逻辑回归算法（二元分类以及多元分类）

    一般用于需要明确输出的场景，如某些事件的发生（预测是否会发生降雨）。
    通常逻辑回归使用某种函数将概率值压缩到某一特定范围。
    例如，Sigmoid 函数（S 函数）是一种具有 S 形曲线、用于二元分类的函数。它将发生某事件的概率值转换为 0, 1 的范围表示。

    5、决策树算法
    描述对实例进行分类的树形结构。从根节点开始，对实例的某一特征进行测试，得到测试结果，再根据另一特征将实例继续分配到下一子节点。就这样根据不同特征递归测试、分配，直至叶节点，得到最终分类结果。
    通过对一系列问题进行“是/否”的推导，最终实现决策。
    场景：调查和征询病人的近期行为：去过医院或高危聚集地->防护措施是否到位->有病患接触史->是否发烧咳嗽，这四个行为特征来判断是否需要对该人进行核酸检测，从而进一步确诊。

    6、随机森林算法
    随机森林可以看作一个决策树的集合。
    随机森林中每棵决策树估计一个分类，这个过程称为“投票（vote）”。理想情况下，根据每棵决策树的每个投票，选择最多投票的分类。

    8、k-平均算法 K-Means（聚类算法）
    无监督学习算法，为聚类问题提供了一种解决方案。
    K-Means 算法把 n 个点（可以是样本的一次观察或一个实例）划分到 k 个集群（cluster），使得每个点都属于离他最近的均值。重复上述过程一直持续到重心不改变。

    9、朴素贝叶斯算法（分类算法）
    假设属性之间相互独立，使用概率统计的知识对样本数据集进行分类。在属性个数比较多或者属性之间相关性较大时，分类效果不好。
    贝叶斯定理是在掌握与相关事件的先决信息下，计算该事件的概率。
    而在属性相关性较小时，朴素贝叶斯性能最为良好。
    场景：判别垃圾邮件，根据关键词代开，增值税,发票，sep，计算为垃圾邮件的概率进而判断是否为垃圾邮件。

    降维算法
    https://www.bilibili.com/video/BV1wa4y1i7Ui?spm_id_from=333.337.search-card.all.click (opens new window)
    降维是指在限定条件下，降低随机变量个数，得到一组“不相关”主变量的过程。
    主成分分析：通过正交变换将众多具有一定相关性的变量重新组合成一组新的互相无关的综合指标来代替原来的指标，转换后的这组变量叫主成分。数学上处理降维 (opens new window)的一种方法。

    1. 构建原始数据矩阵；
    2. 消除量纲——数据标准化（spss自动计算）；
    3. 建立协方差矩阵（就是相关系数矩阵）；
    4. 求出特征值、特征向量 (opens new window)；
    5. 写出主成分，并计算综合得分，给出现实意义的解释

    主成分分析是利用降维的思想，在损失很少信息的前提下把多个指标转化为几个综合指标的多元统计方法。
    转化生成的综合指标称之为主成分，其中每个主成分都是原始变量的线性组合，且各个主成分之间互不相关，这就使得主成分比原始变量具有某些更优越的性能。

    数学建模：spss软件，
    1、数据标准化及因子分析适用性检验（ＫＯＭ 检验法和 球体检验法），验证是否适合主成分分析
    2、 主成分分析过程 ：建立协方差矩阵、特征值、特征向量 (opens new window)
    3、得出方差贡献率，得出前两个主成分解释了全部方差的84%，计算主成分系数
    X1=人均生产总值+社会消费品零售总额+财政总收入+城镇居民人均纯收入、城镇居民人均可支配收入等国民收入
    X2=我国进口总额+出口总额关于我国外贸
    4、主成分得分和综合得分
    将标准化后的原始数据带入主成分表达式就可以计算每个特征样本的主成分得分。
    把两个主成分的方差的占比带入可算综合得分
    对中国的经济水平进行主成分分析，并计算11个主要指标的综合得分。
