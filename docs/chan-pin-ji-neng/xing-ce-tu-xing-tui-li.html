    行测-图形推理

    藤藤同学 3/2/2022

    图形推理（位置规律、样式规律、属性规律、特殊规律）

    # 第一节 位置规律

    题目特征：元素组成相同

    一、平移

    1. 方向：上下、左右、对角线、顺逆时针 2. 常见步数：恒定、递增

    二、旋转、翻转

    旋转：

    1. 方向：顺时针、逆时针 2. 常见角度：30°、45°、60°、90°、180°

    翻转：

    1. 左右翻转：图形沿竖轴对称 2. 上下翻转：图形沿横轴对称

    # 第二节 样式规律

    题目特征：元素组成相似

    一、加减同异

    图形特征：线条重复出现

    1. 相加、相减 2. 求同、求异

    二、黑白运算

    1. 题目特征：图形轮廓和分割区域相同，不同区域“黑白”颜色不同，且黑块数 量不成规律 2. 解题技巧：按照对应位置进行“黑白”相加运算，将所得规律应用于所求图形

    # 第三节 属性规律

    题目特征：元素组成不相同、不相似

    一、对称性

    1. 轴对称（沿着线折叠后，线两边的部分能够完全重合）
    2. 中心对称（图形旋转 180°后和原图形完全重合）
    3. 轴对称 + 中心对称：有两条相互垂直的对称轴

    二、曲直性

    题目特征：题干图形由全曲线图、全直线图或者既有曲线又有直线的图形构成

    三、开闭性

    题目特征：题干图形由全开放图、全封闭图或半开半闭图构成

    # 第四节 特殊规律

    题目特征：题干每幅图都出现两个或多个封闭图形连在一起

    考点：

    1. 相离：图形分开，没有公共部分
    2. 相交： （1）相交于点 （2）相交于线 （3）相交于面：相交面的形状

    二、功能元素

    观察功能元素对其他图形的标记作用

    # 第五节 数量规律

    题目特征：元素组成不相同、不相似且无属性规律

    一、面数量 特征图：图形被分割，封闭面明显

    二、线数量 特征图：

    1. 直线数：多边形、单一直线 2. 曲线数：曲线图形（全曲线图、圆、弧）

    一笔画：

    1. 线条之间全部连通

    2. 奇点数为 0 或 2 个 奇点：发射出奇数条线的点 注：所有的端点都是奇点，数奇点时要数上端点

    ​

    多笔画：

    连通图笔画数 = 奇点数 ÷2（任何图形的奇点数一定为偶数） 常见笔画数特征图： 五角星、“日”、“田”、“日”和“田”的变形图、圆和圆相切与相交、多端点图形

    三、点数量 特征图：线条和线条交叉明显

    四、素数量

    1. 元素种类

    2. 元素个数
